# **Minesweeper**
A simple minesweeper game that runs in the console

## **Installation**

1. **Clone or Download the repository**
   ```bash
   git clone https://github.com/Acrenos/Minesweeper.git
   cd Minesweeper

2. **Compile the Program**
    ```bash
    javac src/*.java

3. **Run the Program**
    ```bash
    java src/Main.java
